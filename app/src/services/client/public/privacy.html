Privacy

Definitions


Privacy

The ability to selectively reveal yourself.


Personally Identifiable Information

Data that can be used to identify you, such as your name, ip address, email address, phone number etc.

All personally identifiable data is end-to-end encrypted on cypherpost servers.


End-to-end encryption

The encryption of your data using a key which is shared only with intended recipients; allowing only the recipients to view the data.

Implementation

Cypherpost servers will not accept plain-text posts.

Public Data

All data flowing through cypherpost clients and servers are end-to-end encrypted with the exception of:

  • Username(optional)
  • Identity PubKey
  • Badges(Reputation)

This is the bare minimum required to allow users to discover other users and build a trusted network.

Private Data

  • Posts
  • Key Store

Data Analytics

We encourage users to always use either a VPN or Tor to prevent IP tracking since server handling of public data cannot be verified.


IP Address, username, xpub constitues your public identity on cypherpost.
